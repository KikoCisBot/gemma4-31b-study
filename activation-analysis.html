Key findings

  • Layer 0 dominates all categories (embedding → first transform)
  • Layer 28 is the most selective: activates 0.99× for Chinese but only 0.77× for tool-calling — it differentiates natural language from structured output
  • Tool-calling uses fewer high layers (50-59) than any other category — JSON structure resolves earlier in the network. This explains why IQ1_M preserves BFCL (86.75%) while destroying code (21%)
  • Spanish has the highest total delta (123.0 on layer 0) — the model works harder for non-English languages
  • Chinese has the lowest total (98.7) but highest relative activation on layers 28, 59 — different processing pattern
  • Code (Python/Rust) and Math share very similar activation profiles — the model uses the same circuits for both
